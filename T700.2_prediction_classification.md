## 4. Critical Dependency: The Baseline Problem

Every threshold-excess prediction takes the form:

```
SFH_true if Measured > Baseline + δ
SFH_false if Measured ≤ Baseline
```

The prediction is falsifiable **only if Baseline is independently fixed**.

| Baseline Source | Vulnerability |
|----------------|---------------|
| Independent physics (Bell, GP, QCD) | Low — baseline is settled science |
| Asserted number with no citation | **High** — author can adjust baseline to match any measurement |
| Asserted with general reference but specific number not derived | **Medium** — reference supports the concept, not the exact value |

**Impact:** 13 of 18 predictions have High or Medium baseline vulnerability. The 3 predictions with Low vulnerability (N35, N36, N46) test standard physics, not SFH — any deviation would already be a discovery in standard science. SFH adds nothing to those measurements.
