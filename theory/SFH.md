# Sentient First Hypothesis (SFH)

## Core Claim

Interaction is primary. Objects are stable interaction patterns, not independently existing entities.

## Tenets

### 1. Interaction precedes identity

What we call "objects" are temporarily stabilized patterns of interaction within a dynamical system. Persistence of interaction structure creates the appearance of identity. When interaction structure dissolves, the "object" ceases to exist.

### 2. Knowing precedes thinking

Primitive forms of interaction-based "knowing" (sensing, responding, coupling) are more fundamental than symbolic representation. Abstract thought and language are late-stage compression mechanisms built on top of this more primitive interaction layer.

### 3. Structure is dynamical

Structure does not exist independently of the dynamics that produce it. What appears as "structure" is a metastable pattern of ongoing interaction — a standing wave, not a carved statue.

### 4. Emergence is relational

Novel properties emerge from patterns of interaction between components, not from the components themselves. The interaction is the fundamental unit, not the interacting parties.

## Contrast with Object-First Realism

| Aspect | Object-First | Interaction-First |
|--------|-------------|-------------------|
| Primitive | Objects with properties | Interaction patterns |
| Identity | Intrinsic to object | Stabilized interaction |
| Change | Object changes state | Pattern transforms |
| Knowledge | Symbolic representation | Embodied coupling |
| Emergence | From objects | From interaction |

## Relationship to SGP

SFH is the ontological position. SGP is the mathematical framework that operationalizes SFH by providing:
- A metric for quantifying interaction structure (coherence)
- Operators for tracking interaction dynamics
- Methods for detecting emergent structure without assuming object boundaries
