# T300.1 — Master Bibliography: Sensitivity of Life-Permitting Structure to Fundamental Parameters

**Program:** T300 — Expanded Analysis from First Principles
**Phase:** 1 (Literature Corpus Expansion)
**Document purpose:** Build a comprehensive, balanced bibliography of 54 sources covering pro-fine-tuning, skeptical, anthropic, multiverse, naturalness, stellar nucleosynthesis, nuclear physics, chemistry, and observer-selection literature.
**Neutral framing (from T300 directive):** This project investigates *"what modern physics actually implies about the sensitivity of life-permitting structure to changes in fundamental parameters"* — not "how fine-tuned is the universe."
**Cross-reference:** T200.1 provides the original 17-source survey with detailed anthropic bounds per constant. This document expands that corpus and adds `stance` and `evidence_type` fields for bias analysis.
**Audit status (T300.1A):** 54 sources — 37% supportive / 26% skeptical / 37% neutral. All T300.1A audit gaps closed: Collins (2009), White (2000), Hossenfelder (2018), McGrew et al. (2001), Piran & Jimenez (2023), Carroll (2019) added. Audit passed. Proceeding to T300.2.

## How to Read This Document

Each entry has a uniform structure:

1. **ID** — Sequential number for cross-referencing
2. **Citation** — Full reference with DOI/arXiv
3. **Keywords** — Main topics addressed
4. **Main Claim** — The source's central thesis relevant to parameter sensitivity
5. **Constants Varied** — Which fundamental parameters are analyzed
6. **Methodology** — How the analysis was performed
7. **Life Criterion** — Definition of "life-permitting" (if applicable)
8. **Probability Framework** — Probabilistic reasoning used (if any)
9. **Known Criticisms** — Major objections from other sources
10. **Stance** — `supportive` / `skeptical` / `neutral`
11. **Evidence Type** — `analytic` / `simulation` / `review` / `philosophical`

### Lexicon

| Term | Meaning |
|------|---------|
| **Stance: supportive** | Argues that parameter sensitivity implies fine-tuning, anthropic selection, or design |
| **Stance: skeptical** | Argues that parameter sensitivity is overstated, misunderstood, or not evidence for anything |
| **Stance: neutral** | Provides review, methodology, or bounds without advocating a position |
| **Evidence type: analytic** | Derives bounds from mathematical physics, scaling relations, or analytic models |
| **Evidence type: simulation** | Uses numerical simulation (BBN codes, stellar models, N-body) to compute bounds |
| **Evidence type: review** | Synthesizes existing literature without new calculations |
| **Evidence type: philosophical** | Addresses conceptual foundations, logic, or interpretation |
