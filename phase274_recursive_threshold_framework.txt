SECTION A: RECURSIVE STABILITY THRESHOLD DEFINITION
================================================================================

OPERATIONAL DEFINITION:
Recursive stability threshold is the critical depth where propagation
geometry transitions from stable to unstable, characterized by
localization fracture, class destabilization, or saturation collapse.

MATHEMATICAL CHARACTERIZATION:

Threshold T exists where:
  For d < T: stable behavior (|ΔP| < 0.02, |T_d - T_0| < 0.02)
  For d >= T: unstable behavior (|ΔP| >= 0.02 OR |T_d - T_0| >= 0.02)

OBSERVED THRESHOLDS:
- Localization fracture: d = 1.0
- Class destabilization: d = 2.0
- Saturation collapse: d = 0.25
- Stable deformation: d < 1.0

================================================================================
SECTION B: LOCALIZATION FRACTURE DEFINITION
================================================================================

OPERATIONAL DEFINITION:
Localization fracture occurs when |ΔP(k=1)| >= 0.02,
indicating loss of confinement at high coupling.

MATHEMATICAL CHARACTERIZATION:

Localization preserved: |ΔP(k=1)| < 0.02
Localization fractured: |ΔP(k=1)| >= 0.02

OBSERVED BEHAVIOR:

depth=0.00: |ΔP| = 0.013 → STABLE_DEFORMATION
depth=0.25: |ΔP| = 0.006 → STABLE_DEFORMATION
depth=0.50: |ΔP| = 0.000 → STABLE_RECURSION
depth=0.75: |ΔP| = 0.003 → STABLE_RECURSION
depth=1.00: |ΔP| = 0.043 → LOCALIZATION_FRACTURE
depth=1.25-2.00: |ΔP| = 0.043 → LOCALIZATION_FRACTURE

THRESHOLD: d = 1.0

================================================================================
SECTION C: CLASS DESTABILIZATION DEFINITION
================================================================================

OPERATIONAL DEFINITION:
Class destabilization occurs when |T_d - T_baseline| >= 0.02,
indicating emergence of new topology class.

MATHEMATICAL CHARACTERIZATION:

Class stable: |T_d - T_0| < 0.02
Class destabilized: |T_d - T_0| >= 0.02

OBSERVED BEHAVIOR:

baseline: T_0 = -0.0139

depth=0.00: |T - T_0| = 0.000 → STABLE
depth=0.25: |T - T_0| = 0.002 → STABLE
depth=0.50: |T - T_0| = 0.001 → STABLE
depth=0.75: |T - T_0| = 0.009 → STABLE
depth=1.00: |T - T_0| = 0.013 → STABLE
depth=1.25: |T - T_0| = 0.012 → STABLE
depth=1.50: |T - T_0| = 0.012 → STABLE
depth=1.75: |T - T_0| = 0.012 → STABLE
depth=2.00: |T - T_0| = 0.023 → DESTABILIZED

THRESHOLD: d = 2.0

================================================================================
SECTION D: SATURATION COHERENCE FAILURE DEFINITION
================================================================================

OPERATIONAL DEFINITION:
Saturation coherence fails when k_s exits [0.4, 0.7],
indicating boundary instability.

MATHEMATICAL CHARACTERIZATION:

Coherence preserved: 0.4 <= k_s <= 0.7
Coherence failed: k_s < 0.4 OR k_s > 0.7

OBSERVED BEHAVIOR:

All tested depths show saturation boundary outside [0.4, 0.7].
Coherence failure at d = 0.25 (first test point).

THRESHOLD: d = 0.25 (immediate failure)
