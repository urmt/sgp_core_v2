## Summary

Machine-readable database of all numerical fine-tuning bounds extracted from the 54-source corpus. Each row records one (source, constant, bound) triple with exact numerical values, life criterion tier, methodology, confidence, and contestation status.

## Column Definitions

| Column | Description |
|--------|-------------|
| `source_id` | Bibliography entry number from T300.1 |
| `citation` | Short author+year reference |
| `constant` | Parameter varied (α, α_s, μ, α_G, v/M_Pl, Λ, etc.) |
| `bound_type` | `upper`, `lower`, `two-sided`, `none` (allowed), or `no_constraint` |
| `bound_low` | Lower bound in natural units (or exponent for Λ) |
| `bound_high` | Upper bound in natural units (or exponent for Λ) |
| `observed_value` | Measured value of this constant |
| `bound_as_fraction` | Human-readable relative bound |
| `life_criterion` | Minimal life criterion used for this bound |
| `tier` | 1-4 (1=robust, 2=moderate, 3=specific, 4=vague) |
| `variation` | `OaA` (one-at-a-time), `Multi-parameter`, `Simulation`, `Correlation`, or philosophical |
| `methodology` | Brief description of calculation method |
| `confidence` | Firm / Moderate / Weak / Low / Skeptical / NA |
| `contested_by` | Source IDs that challenge this bound |
| `notes` | Caveats, cross-references, and qualifiers |

## Key Findings

### Constants with the strongest bounds (Tier 1–2, Firm, uncontested)
- **α_s**: Strong lower bound (deuterium binding) from Barrow & Tipler
- **α**: Upper bound for pair production, lower bound for molecular binding (Barrow & Tipler)
- **Λ**: Upper bound from Weinberg, confirmed by Tegmark & N-body
- **v/M_Pl**: Two-sided from Agrawal et al., confirmed by Hogan

### Constants with wide or contested bounds
- **μ (electron/proton mass ratio)**: Essentially no useful Tier 3 bound in literature — **a genuine gap**
- **α_G**: Factor ~10⁹ range from Carr & Rees; Adams narrows but still wide
- **α_s (carbon bound)**: Narrowest bound in corpus (0.5%) but contested by MacDonald & Mullan

### Constants with zero useful bound
- **masses**: No bounds in literature; masses are themselves rather than variations
- **number of dimensions**: Insufficient for anthropic bound extraction
- **initial conditions**: Penrose bound is the only one — philosophically disputed

### Exceptional rows
- **Stenger Monte Carlo** (id=39): Only entry using direct Monte Carlo; labeled "maximally permissive" — useful as outer bound only
- **Harnik weakless universe** (id=16): Unique "no constraint" row; argues weak interaction irrelevant to life
- **Damour & Donoghue** (id=24): Correlation analysis only; shows α-αs joint variation changes effective constraints
- **Penrose entropy** (id=9): Initial condition not constant; included for completeness only

## Critical Gaps (carried forward to T300.3)

1. **μ bound for Tier 3 (complex molecular chemistry)**: No published bound exists. Every popular claim about μ fine-tuning is unsupported.
2. **α_s Tier 1 diproton claim**: Davies (2%) refuted by MacDonald & Mullan (50%) — key destruction result
3. **Joint variation**: Only 1 source (Damour & Donoghue) explores α-αs correlation; all others assume independent variation
4. **Simulation evidence**: Only 11% of bounds from simulation; majority is analytic scaling
