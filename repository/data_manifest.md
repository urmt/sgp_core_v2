Data Manifest
=============

Source Data
-----------
MEC FRtensor files (21 recordings):
  Source: Gardner et al. (2021), Dryad doi:10.5061/dryad.9s4mw6mh0
  Format: float64 numpy arrays (.npy), shape (T, N) or (trials, time_bins, N)
  Contents: firing rate tensors, binned at 20ms, Gaussian-smoothed σ=40ms
  Recordings: Calais, Goa, Kerala, Mumbai cohorts

Preprocessing Parameters
------------------------
- Bin width: 20 ms
- Gaussian smoothing σ: 40 ms (variable in 015C)
- Normalization: z-score (zero mean, unit variance per neuron)

Analysis Parameters
-------------------
- Spectral decay α: polyfit to eigenvalues 2 through N/2 of correlation matrix
- Participation ratio: PR = (Σλ)² / Σλ²
- Manifold closure LC: log10(∥X - X₉₅∥² / ∥X∥²)
- Mahalanobis distance: covariance-aware ensemble separation
- Permutation tests: n=5000 label shuffles for p-value estimation
- Bootstrap CIs: n=5000 resamples for 95% confidence intervals
