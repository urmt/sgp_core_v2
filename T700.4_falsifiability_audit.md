# T700.4 — Falsifiability Audit

**Status:** Complete
**Method:** For each prediction, determine whether an empirical observation could prove it false. Apply Popperian criterion: a hypothesis is scientific only if it is falsifiable.

---

## 1. Falsifiability Criteria

| Status | Definition | Meaning |
|--------|------------|---------|
| **Falsifiable** | Specific observable outcome that would disprove the prediction | The prediction makes a clear, testable claim |
| **Conditionally Falsifiable** | Falsifiable IF a separate parameter (baseline) is independently fixed | Currently vulnerable to baseline-adjustment defense |
| **Unfalsifiable** | No conceivable observation disproves it | Not a scientific claim |

---

## 2. The Structural Challenge

Every SFH-SGP prediction has this logical form:

> **If Measured > Baseline + δ, then SFH is confirmed.**

The falsification condition is:

> **If Measured ≤ Baseline + δ, then SFH is NOT confirmed (but not necessarily disproven).**

This is **asymmetric confirmation**: the prediction can be confirmed but never truly falsified, because the proponent can always argue:

1. "The baseline was set too high"
2. "The measurement method didn't capture the SF effect"
3. "The SF effect is present but below δ for this particular system"

Without an independent SFH-SGP derivation of Baseline and δ, the claim is **unfalsifiable** regardless of how precise the measurement is.

## 5. Root Cause

The falsifiability problem is structural, not incidental. All 15 quantitative predictions share the same form:

```
C = Measured / Baseline
SFH confirmed if C > threshold
```

Since SFH-SGP provides no method to derive Baseline from its axioms, Baseline is a **free parameter**. A theory with N free parameters can always be adjusted to fit any dataset.

For SFH-SGP to become falsifiable, it would need to do one of:

1. **Derive Baseline from SFH-SGP axioms** — e.g., "the fiber bundle structure group G = SO(N) for neural systems implies Kmat = 0.7." This has never been done.

2. **Predict a relationship between two independent measurements** — e.g., "if C for neural systems exceeds 1.3, then C for social networks must also exceed 1.3, and the ratio must be constant." The book treats each domain independently.

3. **Predict a new phenomenon not already studied by standard science** — e.g., a specific cross-scale correlation signature that can be searched for in existing survey data.

Without one of these, SFH-SGP remains an interpretive framework applied to existing measurements, not a source of falsifiable predictions.
