================================================================================
METRIC DEFINITIONS
================================================================================

HIERARCHY_SURVIVAL:
- Preservation of hierarchy under perturbation
- Range: [0, 1]
- High values indicate robust hierarchy
- Values > 0.70 indicate surviving hierarchy

METRIC_SENSITIVITY:
- Instability under metric changes
- Range: [0, 1]
- Low values indicate metric robustness
- Values < 0.30 indicate bounded sensitivity

TOPOLOGY_DEPENDENCE:
- Dependence on sector arrangement
- Range: [0, 1]
- Low values indicate topology independence

NULL_MODEL_DISTANCE:
- Separation from randomized systems
- Range: [0, 1]
- High values indicate non-random structure
- Values > 0.25 indicate separation from null models

ADVERSARIAL_SURVIVABILITY:
- Persistence under adversarial noise
- Range: [0, 1]
- Values > 0.70 indicate robust survivability

================================================================================
CLASSIFICATION CRITERIA
================================================================================

ROBUST:
- hierarchy_survival > 0.80
- metric_sensitivity < 0.20
- adversarial_survivability > 0.80

WEAKLY_ROBUST:
- hierarchy_survival > 0.70
- metric_sensitivity < 0.25
- adversarial_survivability > 0.70

METRIC-SENSITIVE:
- metric_sensitivity > 0.25
- hierarchy_survival > 0.65

TOPOLOGY-DEPENDENT:
- topology_dependence > 0.25
- hierarchy_survival > 0.65

COLLAPSING:
- hierarchy_survival < 0.65
- OR adversarial_survivability < 0.60
