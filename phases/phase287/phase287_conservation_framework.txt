SECTION 1: OPERATIONAL DEFINITIONS
================================================================================

A. CONSERVATION CONSTRAINT
--------------------------
Definition: A relational quantity exhibiting bounded variance
under recursive operator evolution.

Operationalization:
- Compute mean_variance for each sector across depths
- Compute variance_variance = Var(mean_variance) across depths
- Bounded if variance_variance < 0.01

B. PROPAGATION INVARIANT
------------------------
Definition: A structure remaining statistically stable across
recursive depth and topology variation.

Operationalization:
- Track sector persistence rates across depths
- Stable if persistence rate remains > 0.30
- RG correlation test for scale invariance

C. SECTOR-STABLE QUANTITY
-------------------------
Definition: A measurable property preserved preferentially
within a persistence sector.

Operationalization:
- Compute persistence_rate per sector
- SECTOR_STABLE if persistence > 0.50
- SECTOR_WEAK if 0.30-0.50
- SECTOR_UNSTABLE if < 0.30

D. RG-STABLE INVARIANT
-----------------------
Definition: A conserved relational quantity surviving
coarse-graining.

Operationalization:
- Compute fine-scale metrics
- Compute coarse-scale (RG block) metrics
- Correlation test: > 0.90 = RG_STABLE

E. METASTABLE CONSERVATION
---------------------------
Definition: Approximate preservation despite local fluctuations.

Operationalization:
- Track persistence across all depths
- Nonzero retention = METASTABLE
- Zero retention = NONCONSERVING
