================================================================================
METRIC DEFINITIONS
================================================================================

REDUNDANCY_RETENTION:
- Persistence preserved through redundancy
- Range: [0, 1]
- High values indicate redundant stability
- Values > 0.75 indicate redundant-stable structure

COMPENSATION_STRENGTH:
- Recovery via alternate pathways
- Range: [0, 1]
- High values indicate strong compensation
- Values > 0.70 indicate effective compensation

FAILURE_CONTAINMENT:
- Bounded degradation spread
- Range: [0, 1]
- High values indicate contained failure
- Values > 0.70 indicate bounded propagation

DISTRIBUTED_SUPPORT_GAIN:
- Stabilization gain from redundancy
- Range: [0, 1]
- Positive values indicate cooperative benefit

RECURSIVE_SURVIVABILITY:
- Retained persistence under repeated disruption
- Range: [0, 1]

================================================================================
CLASSIFICATION CRITERIA
================================================================================

REDUNDANT-STABLE:
- redundancy_retention > 0.80
- compensation_strength > 0.75
- failure_containment > 0.75

WEAKLY_REDUNDANT:
- redundancy_retention > 0.70
- compensation_strength > 0.65
- failure_containment > 0.65

FAILURE-DEGRADING:
- redundancy_retention > 0.65
- compensation_strength > 0.55
- failure_containment > 0.55

COMPENSATION-PRESERVING:
- compensation_strength > 0.80
- redundancy_retention > 0.70
- failure_containment > 0.70

COLLAPSING:
- redundancy_retention < 0.65
- OR failure_containment < 0.50
- OR compensation_strength < 0.50
