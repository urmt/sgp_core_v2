# Phase 411 Analysis: Synthetic Correspondence

## Verdict: SYNTHETIC-CORRESPONDENCE-DEGRADING (2/5)

The SFH-SGP hierarchy does NOT generalize to independent synthetic
systems. Correspondence is statistically indistinguishable from null.

## What This Means

The SFH-SGP 7-sector hierarchy is framework-specific. It does not
appear in:
- Cellular automata with different computational rules
- Graph propagation on random topologies
- Agent interaction systems
- Symbolic recursive L-systems
- Tensor network contraction schemes
- Evolutionary optimization dynamics

This is a NEGATIVE but scientifically VALUABLE result. It establishes
an empirical boundary on the framework's generality.

## Post-Hoc Observations (NOT hypothesis tests)

1. **Consistent weak positive for 4/6 systems.** CA, GPS, AIS, and SRS
   all show rho > 0 (though not significant). This is a directional
   hint but not evidence.

2. **Evolutionary systems invert the hierarchy.** Preservation-focused
   evolutionary mechanisms (fitness sharing, speciation) rank LOW in
   SFH-SGP (neutral/anarchic sectors) but produce the most structured
   populations. This suggests the SFH-SGP hierarchy prioritizes
   operator richness over structural preservation.

3. **The framework may capture P-A-N-specific emergence rather than
   universal emergence.** The hierarchy reflects the specific
   mathematical structure of the projection-antisymmetry-neutrality
   operator algebra.

## Methodological Integrity

The methodology was strict:
- Pre-defined config-to-sector mapping (no post-hoc fitting)
- Null-controlled via 10,000 random permutations per system
- Independent emergence computation per system (no shared math)
- If correspondence had appeared, it would have been genuine

The negative result is trustworthy because of this rigor.
