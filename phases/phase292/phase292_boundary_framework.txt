================================================================================
SECTION 1: OPERATIONAL DEFINITIONS
================================================================================

A. RELATIONAL BOUNDARY
-----------------------
Definition: A statistically stable separation region between
recursive propagation regimes.

Operationalization:
- Compare trajectory distributions between sector pairs
- Boundary strength = separation between centroids / combined dispersion
- Strong boundary = large separation, low overlap

B. BOUNDARY STABILITY
----------------------
Definition: Persistence of sector separation structure across
recursive depth.

Operationalization:
- Track boundary_strength across depths
- Stable if variance < threshold
- Unstable if variance increases

C. TRANSITION INTERFACE
------------------------
Definition: A recursive configuration region mediating sector
transitions.

Operationalization:
- Identify trajectory points near sector boundaries
- Compute interface dispersion
- Narrow interface = strong boundary

D. SECTOR SEPARABILITY
------------------------
Definition: Degree to which recursive trajectories remain
partitioned between sectors.

Operationalization:
- separability = 1 - cross_entropy(sector_A, sector_B)
- Higher separability = better partition
- Increase with depth = separability learning

E. RG-STABLE BOUNDARY
----------------------
Definition: Boundary structure preserved under coarse-graining.

Operationalization:
- Compute fine-scale boundary at multiple depths
- Compare coarse-scale (RG block) boundary
- Correlation > 0.90 = RG_STABLE

SHARP:

WEAKLY_SHARP:

DIFFUSE:
