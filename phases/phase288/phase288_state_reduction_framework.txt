SECTION 1: OPERATIONAL DEFINITIONS
================================================================================

A. EFFECTIVE STATE REDUCTION
-----------------------------
Definition: Decrease in accessible relational configuration
volume under recursive composition.

Operationalization:
- Compute accessible_volume = sector_count / N_STATES
- Compare across depths
- Reduction if volume decreases

B. ACCESSIBLE CONFIGURATION SPACE
-----------------------------------
Definition: The set of reachable embedding-sector configurations
under validated operator evolution.

Operationalization:
- Track sector distribution
- Compute state embeddings
- Measure configuration diversity

C. RECURSIVE COMPRESSION
-------------------------
Definition: Progressive reduction of accessible configuration
diversity across depth.

Operationalization:
- Track volume ratio across depths
- Ratio < 1 indicates compression
- Ratio > 1 indicates expansion

D. METASTABLE MANIFOLD
-----------------------
Definition: A constrained relational region repeatedly revisited
under recursive evolution.

Operationalization:
- Compute manifold_compactness = 1 / mean_centroid_distance
- Higher compactness = tighter manifold
- Track across depths

E. RG-STABLE REDUCTION
-----------------------
Definition: State reduction structure preserved under
coarse-graining.

Operationalization:
- Compute fine-scale volumes at multiple depths
- Compare coarse-scale (RG block) volumes
- Correlation > 0.90 = RG_STABLE
