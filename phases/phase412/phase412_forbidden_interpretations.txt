# Phase 412: Forbidden Interpretations

## STRICTLY FORBIDDEN

1. **"No excluded systems means every system is SFH-SGP-like."**
   PERIPHERAL means score 0.20-0.49, not CORE. The gap from
   0.55 to 1.00 is the true universality boundary. No system
   besides SFH-SGP itself is close to 1.00.

2. **"H1 failed, so SFH-SGP doesn't require specific constraints."**
   H1 used "excluded" threshold (<0.20) which was too strict.
   All 6 non-core systems have significant deficiencies (4-9
   features below 0.50). Specific constraints ARE required.

3. **"Two attractor families means all synthetic systems are one type."**
   Family 2 is a heterogeneous cluster with inter-distances
   0.45-1.40. The six synthetic systems span diverse structures.
   They cluster ONLY because SFH-SGP is even more distant.

4. **"Correspondence gradient r=0.70 proves membership predicts
   emergence transfer."**
   p = 0.08. This is a positive trend with only 7 data points.
   It is suggestive but not significant. Do not treat as proof.

5. **"Boundary stability over 80% means classification is fixed."**
   84.9% mean stability but most systems show 3 different classes
   across perturbations. The boundary is stable in the majority
   case but has meaningful sensitivity.

## PERMITTED INTERPRETATIONS

1. "SFH-SGP is a bounded universality class, its own attractor family."
2. "Six specific features define SFH-SGP universality membership."
3. "No synthetic system is fully excluded; all share some features."
4. "Hierarchy correspondence correlates positively with feature
   membership (r=0.70, p=0.08, n=7)."
5. "The SFH-SGP universality boundary is stable at ~84% under
   reasonable weight perturbations."
