================================================================================
SECTION 1: OPERATIONAL DEFINITIONS
================================================================================

A. LOCAL FIXED STRUCTURE
-------------------------
Definition: A recursively revisited bounded relational configuration region.

Operationalization:
- Identify regions visited multiple times within recursive depth
- Compute retention probability: P(revisit | entry)
- Bounded region = radius < threshold of configuration space
- Higher retention = more fixed structure

B. METASTABLE FIXED REGION
---------------------------
Definition: A locally persistent configuration exhibiting bounded drift
under recursive evolution.

Operationalization:
- Compute drift rate = mean_displacement / depth
- Bounded drift = rate < threshold
- Track region persistence across recursive depths
- Metastable = survives multiple depth iterations

C. GLOBAL FLOW DIVERGENCE
--------------------------
Definition: Large-scale recursive trajectory instability despite local
boundedness.

Operationalization:
- Compute trajectory escape probability
- Escape = displacement exceeds bounded region radius
- Global divergence = high escape rate
- Local bounded = retention within region

D. FIXED-REGION RETENTION
--------------------------
Definition: Persistence probability of remaining within a bounded
relational neighborhood.

Operationalization:
- retention = count(return visits) / count(total visits)
- Track retention across depths
- High retention (>0.5) = fixed structure present

E. RG-STABLE FIXED STRUCTURE
-----------------------------
Definition: Local fixed-region behavior preserved under coarse-graining.

Operationalization:
- Compute fine-scale retention at multiple depths
- Compare coarse-scale (RG block) retention
- Correlation > 0.90 = RG_STABLE
