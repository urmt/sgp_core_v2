================================================================================
METRIC DEFINITIONS
================================================================================

DOMAIN_COHFERENCE:
- Internal persistence consistency within domain
- Range: [0, 1]
- High values indicate coherent domains
- Values > 0.75 indicate stable domains

BOUNDARY_RETENTION:
- Persistence of domain separation across depths
- Range: [0, 1]
- High values indicate stable boundaries
- Values > 0.70 indicate bounded domains

DEGRADATION_PENETRATION:
- Spread of destabilization into domain
- Range: [0, 1]
- Low values indicate bounded penetration
- Values < 0.30 indicate protected domains

PERSISTENCE_DENSITY:
- Concentration of retained organization
- Range: [0, 1]
- High values indicate dense persistence
- Values > 0.70 indicate localized domains

RECURSIVE_DOMAIN_STABILITY:
- Cross-depth domain persistence
- Range: [0, 1]
- High values indicate stable domains
- Values > 0.70 indicate persistent structure

RG_SIMILARITY:
- Fine-grained vs coarse-grained domain preservation
- > 0.90 indicates stable scale invariance

================================================================================
NETWORK-SPECIFIC PREDICTIONS
================================================================================

P-A-N COMPOSITE:
- Strongest domain coherence
- Best boundary retention
- Bounded degradation penetration
- High persistence density
- Forms strongest persistence domain

P-A PAIR:
- Strong domain coherence
- Good boundary retention
- Moderate penetration
- Stable domains
- Strong pair domains

P-N PAIR:
- Moderate domain coherence
- Some boundary degradation
- Partial penetration
- Weakening domains
- Weaker pair domains

A-N PAIR:
- Weak domain coherence
- Low boundary retention
- High penetration
- Collapsing domains
- Weakest pair domains
