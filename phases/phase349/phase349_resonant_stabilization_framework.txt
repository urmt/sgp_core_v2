METRIC DEFINITIONS
================================================================================

RESONANT_GAIN:
- Persistence amplification from synchronized alignment
- Range: [-0.3, 0.3]
- Positive values indicate amplification benefit
- Values > 0.20 indicate strong resonance

SYNCHRONIZATION_RETENTION:
- Retained recursive alignment across depths
- Range: [0, 1]
- High values indicate sustained synchronization
- Values > 0.70 indicate stable alignment

COOPERATIVE_AMPLIFICATION:
- Stabilization increase from coordinated action
- Measures amplification beyond non-resonant networks
- Values > 0.70 indicate strong amplification

DESYNCHRONIZATION_DRIFT:
- Loss from recursive misalignment
- Range: [0, 1]
- Low values indicate bounded desynchronization
- Values < 0.30 indicate good alignment retention

RECURSIVE_RESONANCE:
- Depth persistence of synchronized organization
- Measures resonance survival through recursion
- Values > 0.70 indicate persistent resonance

RG_SIMILARITY:
- Fine-grained vs coarse-grained resonance preservation
- > 0.90 indicates stable scale invariance
