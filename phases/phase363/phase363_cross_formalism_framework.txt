================================================================================
METRIC DEFINITIONS
================================================================================

HIERARCHY_CONSISTENCY:
- Preservation of hierarchy across formalisms
- Range: [0, 1]
- High values indicate formalism-independent hierarchy
- Values > 0.70 indicate consistent hierarchy

FORMALISM_STABILITY:
- Persistence stability under reformulation
- Range: [0, 1]
- High values indicate stable persistence across formalisms

STRUCTURAL_OVERLAP:
- Overlap between formalisms
- Range: [0, 1]
- High values indicate convergent structure
- Values > 0.65 indicate substantial overlap

NULL_FORMALISM_DISTANCE:
- Separation from randomized formalisms
- Range: [0, 1]
- High values indicate non-random structure
- Values > 0.25 indicate separation from null formalisms

RECURSIVE_RETENTION:
- Cross-depth persistence
- Range: [0, 1]
- Values > 0.70 indicate persistent recursion

================================================================================
CLASSIFICATION CRITERIA
