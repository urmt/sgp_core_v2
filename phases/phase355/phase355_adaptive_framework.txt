================================================================================
METRIC DEFINITIONS
================================================================================

ADAPTIVE_RETENTION:
- Persistence preserved during adjustment
- Range: [0, 1]
- High values indicate stable adaptation
- Values > 0.75 indicate adaptively stable structure

RECOVERY_EFFICIENCY:
- Restoration after perturbation
- Range: [0, 1]
- High values indicate efficient recovery
- Values > 0.70 indicate strong recovery

ADAPTIVE_DRIFT:
- Organizational distortion from adaptation
- Range: [0, 1]
- Low values indicate bounded drift
- Values < 0.30 indicate controlled adaptation

COOPERATIVE_ADAPTIVE_GAIN:
- Stabilization gained through adaptation
- Range: [0, 1]
- Positive values indicate cooperative benefit

RECURSIVE_ADAPTIVE_PERSISTENCE:
- Cross-depth adaptive retention
- Range: [0, 1]

================================================================================
CLASSIFICATION CRITERIA
================================================================================

ADAPTIVELY_STABLE:
- adaptive_retention > 0.80
- recovery_efficiency > 0.75
- adaptive_drift < 0.20

WEAKLY_ADAPTIVE:
- adaptive_retention > 0.70
- recovery_efficiency > 0.65
- adaptive_drift < 0.30

ADAPTATION-DEGRADING:
- adaptive_retention > 0.65
- recovery_efficiency > 0.55
- adaptive_drift < 0.40

RECOVERY-PRESERVING:
- recovery_efficiency > 0.80
- adaptive_retention > 0.70
- adaptive_drift < 0.25

COLLAPSING:
- adaptive_retention < 0.65
- OR adaptive_drift > 0.45
- OR recovery_efficiency < 0.50
