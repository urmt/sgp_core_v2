| **C: Interaction (C+friction+synergy)** | (a) Full core model R² = 0.523 (ΔC), 0.565 (restoration), 0.345 (τ_rec). (e) Interaction sign stable: 99.9% positive for ΔC, 99.7% negative for restoration, 98.8% positive for τ_rec across 5000 bootstrap samples. (f) C adds ΔR² = 0.129 (ΔC), 0.271 (restoration), 0.126 (τ_rec) beyond friction alone. | **D: Thermometer (C is non-causal proxy)** | (a) Parsimonious: C and friction are so entangled (r = −0.84) that C may carry no unique causal information.
