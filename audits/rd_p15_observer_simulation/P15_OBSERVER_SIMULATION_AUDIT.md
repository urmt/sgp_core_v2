## 4. Analysis

### 4.1 Critical Finding: Zero Organizational Invariants

**No organizational structures survive observer diversification.**

All structures are observer-coupled:
- Pipeline A claims structure only at high N
- Pipeline B claims structure at all N
- Pipeline C claims structure only at high N with warnings
- Pipeline D claims structure at moderate N
- Pipeline E claims structure at moderate N

### 4.2 Pipeline Divergence

| Pipeline | Characterization | Behavior |
|----------|------------------|----------|
| A | Conservative | Underclaims, frequent warnings |
| B | Permissive | Overclaims, no warnings |
| C | Adversarial | Maximum sensitivity, frequent warnings |
| D | Independent | Moderate claims, independent language |
| E | Stabilizability-focused | Convergence-focused classification |

### 4.3 Warning Activation Asymmetry

Warning activation is highly pipeline-dependent:
- Pipeline B never activates warnings (0/10)
- Pipeline C always activates warnings (10/10)
- Other pipelines show intermediate behavior

This means **admissibility culture is part of the measurement system**.

### 4.4 AM Domain

AM shows limited divergence due to N=2 constraint. Most pipelines except B claim no structure.

## 5. Key Findings

1. **Zero organizational invariants across all pipelines.** This is a scientifically valuable finding.

2. **All structures are observer-coupled.** No structure was claimed by all 5 pipelines at any N.

3. **Warning activation is pipeline-dependent.** Admissibility culture is part of the measurement system.

4. **Pipeline behavior matches design intent.** Conservative underclaims, permissive overclaims, adversarial maximizes sensitivity.

## 6. Implications

1. **The program's organizational structures are observer-dependent.** This is a major finding.

2. **Admissibility culture shapes results.** Different thresholds produce different organizational recoveries.

3. **Warning activation is not universal.** Some pipelines never activate warnings, others always do.

4. **P15 operationalizes skepticism successfully.** The audit detects observer-coupled structures.

## 7. Interpretation

The highest-value outcome has been achieved:

> **Some structures survive observer independence while others collapse.**

In this case: **all structures collapse under observer diversification.**

This means:
- The program's organizational findings are observer-coupled
- Admissibility culture is part of the measurement system
- Warning activation is not universal

## 8. Limitations

1. **Simulated pipelines:** Current analysis uses simulated pipeline behavior. Real observers may differ.

2. **Limited domains:** AM has only 2 trajectories.

3. **Pipeline design:** Pipeline characteristics are predetermined, not independently derived.

4. **No real external observers:** This is a simulation, not a true blind replication.
