# RD-CI RULE

**Source:** Research Director (RD-WELL.8A authorization)
**Status:** ACTIVE RULE
**Date:** 2026-06-17

## Statement

**"No cross-domain comparison may be promoted without uncertainty estimates."**

Confidence intervals are now mandatory for all cross-domain comparisons. A single point estimate is no longer sufficient.

## Required Format

For any cross-domain comparison:
- Report mean ± std
- Report 95% CI (1.96 × std / √N)
- Report N (sample size)
- Note replication status (preliminary, replicated, fully replicated)

## Rationale

The project is entering the era of uncertainty quantification. Values without uncertainty estimates cannot be compared across domains.

## Relationship to Other Rules

- **RD-PROVENANCE RULE:** Every measurement must point to the audit that generated it.
- **RD-CI RULE:** Every cross-domain comparison must include uncertainty estimates.
