# RD-OBSERVER.1 — Observer Bias Subtraction Framework

**Goal:** Estimate observer dependence of explanations.

**Method:** Three levels of increasing rigor.

---

## Level 1: Independent Decompositions

**Description:** Different researchers, different ontologies, same data. Compare results.

**Gold standard:** Multiple independent research groups analyzing the same data with different frameworks.

**Current status:** Not yet implemented.

**Implementation:**
1. Share data with independent researchers
2. Each researcher applies their own framework
3. Compare survivors
4. Overlap becomes stronger evidence

---

## Level 2: Blind Analysis

**Description:** Hide study names, conclusions, ontology words. Analyze raw data only.

**Current status:** Partially implemented in RD-HIST.2A (blind coding).

**Implementation:**
1. Remove all identifying information from data
2. Remove all theoretical vocabulary
3. Analyze raw data with open coding
4. Compare results across independent coders

**Already done:**
- RD-HIST.2A: 20 studies, blind coding, 5 categories
- RD-HIST.2B: 3 clusterings, assignment stability audit

---

## Level 3: Artificial Alien Researchers

**Description:** Create multiple agents with different priors. Give all of them identical raw data. Compare survivors.

**Current status:** Not yet implemented.

**Implementation:**
1. Create agents with different priors:
   - Agent A: Reductionist physicist
   - Agent B: Biologist
   - Agent C: Information theorist
   - Agent D: Process philosopher
   - Agent E: Sentience-first ontology
2. Give all agents identical raw data
3. Each agent applies their own framework
4. Compare survivors
5. Overlap becomes much stronger evidence

**Formal statement:**

> Evidence Strength ∝ Observer Invariance

---

## Relationship to Physics

Physics already does this:
- Gauge invariance
- Coordinate invariance
- Renormalization

RD may be moving toward:
- **Observer invariance of explanations**

---

## Relationship to Original Intuition

Original intuition:

> What can persist in sentience?

Revised intuition:

> What can persist across observers?

These are different questions.

RD may have been slowly drifting toward the second one all along.

---

## Standing Rule 31 (RD-OBSERVER.1)

> When evaluating any survivor, ask: Would this survivor be found by an independent researcher with different priors? The overlap between independent observers becomes the evidence.
